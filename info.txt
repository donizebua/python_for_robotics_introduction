##Modul Python Bawaan (Built-in Modules)
math 
        – Modul untuk fungsi matematika dasar seperti sqrt(), sin(), cos(), dan lain-lain.
os 
        – Modul untuk berinteraksi dengan sistem operasi, seperti file dan direktori.
sys 
        – Modul untuk mengakses dan mengelola parameter dan fungsi sistem.
datetime 
        – Modul untuk bekerja dengan tanggal dan waktu.
random 
        – Modul untuk menghasilkan angka acak dan memilih elemen acak dari urutan.
json 
        – Modul untuk bekerja dengan data dalam format JSON.
re 
        – Modul untuk ekspresi reguler (regex).
time 
        – Modul untuk mengakses fungsi waktu, seperti sleep() dan pengukuran waktu.
collections 
        – Modul yang menyediakan struktur data tambahan seperti Counter, deque, defaultdict.
itertools 
        – Modul untuk manipulasi iterator, menghasilkan kombinasi, permutasi, dan kombinasi.
csv 
        – Modul untuk membaca dan menulis file CSV.
sqlite3 
        – Modul untuk berinteraksi dengan database SQLite.
functools 
        – Modul untuk fungsi terkait pemrograman fungsional, seperti dekorator dan fungsi pembantu.
socket 
        – Modul untuk pemrograman jaringan (sockets).
subprocess 
        – Modul untuk menjalankan proses baru, berinteraksi dengan proses, dan mengeksekusi perintah sistem.

Pandas 
        – Library untuk manipulasi dan analisis data, terutama untuk data tabel (DataFrame). Modul utama: pandas
Matplotlib

Seaborn

SciPy

TensorFlow 
        – Library untuk machine learning dan komputasi numerik, terutama untuk deep learning. Modul utama: tensorflow
Keras 
        – Library untuk neural networks dan deep learning, bekerja di atas TensorFlow. Modul utama: keras
Flask 
        – Microframework untuk pengembangan aplikasi web yang ringan dan fleksibel. Modul utama: flask
Django 
        – Framework web tingkat tinggi untuk pengembangan aplikasi web yang lebih besar dan kompleks. Modul utama: django
PyTorch 
        – Library untuk machine learning dan deep learning, mirip dengan TensorFlow. Modul utama: torch
OpenCV 
        – Library untuk pengolahan gambar dan visi komputer. Modul utama: cv2
Scikit-learn

Requests

BeautifulSoup 
        – Library untuk web scraping, mengambil data dari halaman web. Modul utama: bs4
SQLAlchemy 
        – Library untuk ORM (Object Relational Mapping), memudahkan interaksi dengan database SQL. Modul utama: sqlalchemy
Pillow

Scrapy 
        – Framework untuk web scraping yang lebih kuat dan terstruktur. Modul utama: scrapy
Boto3

PyGame 
        – Library untuk pengembangan game 2D. Modul utama: pygame
pytest

*pip adalah manajer paket Python yang digunakan untuk menginstal dan mengelola pustaka eksternal (dan juga pustaka dari repositori Python Package Index (PyPI)). Dengan menggunakan pip, kamu bisa menginstal pustaka eksternal ke dalam lingkungan Python kamu. Contoh perintah pip untuk menginstal pustaka eksternal:

pip install numpy  # Menginstal pustaka eksternal NumPy

*Untuk memeriksa path yang digunakan Python untuk mencari pustaka, kamu bisa menggunakan kode berikut:

import sys
print(sys.path)

*Jika kamu sudah memiliki pustaka dalam bentuk arsip lokal, kamu bisa menginstalnya dengan cara:

pip install path/to/library.tar.gz
pip install path/to/library.whl

*Cara menggunakan PyPI dengan pip: Jika kamu ingin menginstal pustaka dari PyPI, cukup gunakan perintah pip install di terminal atau command prompt:

pip install requests  # Menginstal pustaka 'requests' dari PyPI



The list of assert method

assertNotEqual(a, b)           a != b

assertTrue(x)                  bool(x) is True

assertFalse(x)                 bool(x) is False

assertIs(a, b)                 a is b

assertIsNot(a, b)              a is not b

assertIsNone(x)                x is None

assertIsNotNone(x)             x is not None

assertIn(a, b)                 a in b

assertNotIn(a, b)              a not in b

assertIsInstance(a, b)         isinstance(a, b)

assertNotIsInstance(a, b)      not isinstance(a, b)


python -m unittest discover -> digunakan untuk menguji semua file yang memiliki pola nama "test_*.py" (pola penamaan test default) yang ada dalam current directory.

python -m unittest discover -p "*_test.py" -> sama seperti diatas, hanya saja dengan pola penamaan yang bisa disesuaikan misalnya "*_test.py".
